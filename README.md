## 🎮 How to use

###  Setup

Clone repository
```
git clone git@github.com:victorsantosbrazil/financial-institutions-api.git
```

Enter the project folder
```
cd financial-institutions-api
```

Install dependencies
```
go get .
```

### Run
```
go run main.go
```

## 📌 Tests

To run all tests.
```
go test .
```
