## 🎮 How to use

### Setup

Clone repository

```
git clone git@github.com:victorsantosbrazil/easyfi-accounts-api.git
```

Enter the project folder

```
cd easyfi-accounts-api
```

Install

```
make install
```

### Run

```
make run
```
